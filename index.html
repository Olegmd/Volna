Как сделать так, чтобы приложение слышало весь разговор

Распознавание слушает микрофон. Чтобы оно слышало и клиента из софтфона, звук ПК «заворачивается» в виртуальный микрофон:

VB-CABLE (виртуальный кабель)
  1. Скачайте с vb-audio.com/Cable, установите от имени администратора, перезагрузите ПК.
  2. Выведите звук софтфона (или всего ПК) на CABLE Input.
  3. Чтобы слышать клиента самому: вкладка «Запись» → CABLE Output → Свойства → «Прослушать» → галочка «Прослушивать с данного устройства» → выберите свои наушники.
  4. В Chrome выберите микрофон CABLE Output (Настройки → Настройки сайтов → Микрофон).
Чтобы в транскрипцию попадал ещё и ваш голос с гарнитуры — поставьте бесплатный Voicemeeter (тот же сайт): он смешивает микрофон и звук ПК.
⚠️ Транскрибация — форма записи разговора. Убедитесь, что клиенты уведомляются о записи («разговор может быть записан»), и согласуйте инструмент с руководителем.
AI-режим (опционально)

Без ключа работает локальный суфлёр по ключевым словам — бесплатно и без лимитов. С ключом добавляются умные AI-подсказки по ходу диалога (следуют вашему скрипту) и развёрнутый AI-разбор звонка.

GigaChat (Сбер) — бесплатно, доступен в РФ
  1. Зайдите на developers.sber.ru → войдите через Сбер ID.
  2. «Мои проекты» → «Создать проект» → выберите GigaChat API (тариф для физлиц, Freemium).
  3. В проекте нажмите «Сгенерировать ключ» и скопируйте Ключ авторизации (длинная строка).
  4. Вставьте его ниже, провайдер «GigaChat», нажмите «Сохранить».
Бесплатно даётся 1 млн токенов (~1000 подсказок) без привязки карты, дальше пакеты от ~1300 ₽ за 20 млн. Работает только на сайте (версия на Netlify), не из локального файла.
Google Gemini — бесплатно, но недоступен в подсанкционных регионах
Ключ создаётся на aistudio.google.com без карты. Если сайт пишет про санкции — используйте GigaChat выше.
Anthropic Claude — платно, качество подсказок выше
Ключ создаётся на console.anthropic.com (нужен баланс). Также недоступен в подсанкционных регионах.
Локальная нейросеть (Ollama) — бесплатно, без интернета, без лимитов и ключей
Модель работает прямо на вашем ПК, данные никуда не отправляются. Нужно 8+ ГБ оперативной памяти; на слабом ПК ответы медленнее (несколько секунд). Качество подсказок ниже, чем у GigaChat/Claude, но для суфлёра достаточно.
  1. Скачайте Ollama с ollama.com (Download → Windows), установите.
  2. Откройте командную строку (Win+R → введите cmd → Enter) и выполните: ollama pull qwen2.5:7b-instruct — скачается модель ~4,7 ГБ. Для слабого ПК: ollama pull qwen2.5:3b-instruct.
  3. Разрешите сайту обращаться к Ollama: там же выполните setx OLLAMA_ORIGINS "*" и полностью перезапустите Ollama (значок в трее → Quit, затем запустить снова).
  4. Выберите ниже провайдера «Локальная нейросеть», проверьте название модели, нажмите «Сохранить». Ключ не нужен.
Ключ хранится только в вашем браузере. На бесплатных тарифах облачные провайдеры могут использовать запросы для улучшения своих моделей — не проговаривайте в звонках паспортные и платёжные данные клиентов. Локальная нейросеть данные не отправляет вообще.
